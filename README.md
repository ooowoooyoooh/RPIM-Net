# RPIM-Net: Residual Channel Prior-Driven Interaction Multi-Scale Network for Stereo Image Deraining

**[RPIM-Net: Residual Channel Prior-Driven Interaction Multi-Scale Network for Stereo Image Deraining]**  
## Getting Started

### Prerequisites

- Python 3.11

#### Training

To train the model, run:
